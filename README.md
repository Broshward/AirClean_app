# 📡 ESP32 BluFi Debug & Control App

Универсальный пульт управления для **ESP32**, работающий через протокол **BluFi**. Приложение позволяет настраивать Wi-Fi, отслеживать показания датчиков и изменять сетевые параметры (DHCP/Static IP) без использования проводов.

## ✨ Основные функции
- **Поиск устройств**: Сканирование и фильтрация ESP32 в режиме BluFi.
- **Мониторинг датчиков**: Отображение температуры окружения, чипа и уровня освещенности в реальном времени.
- **Настройка Wi-Fi**: Сканирование доступных сетей и защищенная передача учетных данных (SSID/Pass).
- **Сетевой менеджер**: 
  - Просмотр текущего IP, маски и шлюза.
  - Переключение между DHCP и **Static IP**.
  - Сохранение настроек в энергонезависимую память (NVS) на стороне ESP32.
- **Диагностика**: Встроенный лог событий для отладки команд.

## 🛠 Технологии
- **Frontend**: Flutter (Dart) + `esp_blufi: ^0.1.8`
- **Backend (Hardware)**: ESP32 (ESP-IDF / Arduino)
- **Протокол**: Bluetooth LE (BluFi)

## 🚀 Быстрый старт (Android)

### Требования
1. Установленный **Flutter SDK**.
2. Android-смартфон с поддержкой Bluetooth и GPS.

### Сборка приложения
```bash
# Клонировать репозиторий
git clone https://github.com/Broshward/AirClean_app.git

# Перейти в папку
cd AirClean_app

# Загрузить зависимости
flutter pub get

# Запустить проект (в режиме отладки)
flutter run

# Собрать APK
flutter build apk --release

⚠️ Важные нюансы (Android 10+)
Для работы сканирования Bluetooth на Android обязательно должны быть включены:

    1. Bluetooth (очевидно).
    2. Геолокация (GPS) — это требование безопасности Android для поиска BLE-устройств.
